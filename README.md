# MediBot - Healthcare Chatbot

## 🚀 Project Run Karne Ka Tarika

### Step 1: Dependencies Install Karein
```bash
npm install
```

### Step 2: Development Server Start Karein
```bash
npm run dev
```

### Step 3: Browser Mein Open Karein
```
http://localhost:3000
```
